Software as a service

SaaS is an application or service that can be run over the internet or on the cloud, for example email, customer relationship management, or analytics tools.

Forms, Analytics, Collaboration, Content Management Systems, Management (CRM), more…

Platform as a service

PaaS is a service that allows users to set up a system on the cloud that other services and applications will be designed and built off of, for example an operating system or development environment.

Application Deployment, Components, Unlisted, more…

Infrastructure as a service

IaaS are structures and facilities, provided over the internet, needed for the operation of platforms and applications to work, for example virtual server space, network connections and content delivery networks.

Compute, Storage, Content Delivery Network, Other, more…

Specialist cloud services
